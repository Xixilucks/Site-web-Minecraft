Présentation du jeu


Minecraft est un jeu vidéo de type bac à sable développé par Mojang AB. Le jeu a été créé par Markus Persson (aussi connu sous le pseudonyme Notch) en mai 2009, puis publié en novembre 2011. Le jeu est disponible en version payante et en version gratuite. La version gratuite est limitée en fonctionnalités, mais permet de jouer en ligne avec d'autres joueurs. Le jeu ne cesse d'évoluer et de nouvelles version sortent régulièrement. Dans le jeu on retrouve une quête principale qui est d'abatre l'Ender Dragon ( boss final du jeu ) mais aussi de nombreuse quête secondaire. Le jeu est composé de plusieurs mondes générés aléatoirement, chacun avec ses propres ressources, ses monstres et ses biomes. Le joueur peut se déplacer librement dans ces mondes, et ainsi y construire librement ce qu'il y souhaite.


Présentation du site web


Ce site web a été créé par un groupe de 3 élèves de 1er au lycée Paul Claudel d'Hulst. Ce site web a été créé dans le but de présenter le jeu Minecraft ainsi que les différents mods, serveurs et communautés qui existent autour de ce vats jeu. Ce site web a été coder essentiellement en HTML, CSS et JavaScript. Ce site web regroupe les serveurs, builds ee mods les plus utiles et les plus populaires de la communauté Minecraft. Ce site web est régulièrement mis à jour afin de vous proposer les meilleurs contenus possible.
