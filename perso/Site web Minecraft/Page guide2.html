Présebtation des différents modes de jeux


La survie


=> Dans Minecraft, plusieurs modes de jeux sont mis à votre disposition, le plus connue étant le mode survie. Dans ce mode de jeu, il vous faudra récolter des matériaux pour vous confectionner un équipement digne de ce nom et ainsi être capable de vous défendre contre les monstres une fois la nuit tombée.

=> La prochaine étape est de se construire un chez soi ou l'on se sent bien. Vous pouvez vous insaller où vous voulez et construire n'importe quel bâtiment, à vous de choisir! Une fois que vous serez assez avancé dans votre aventure, vous pourrez automatiser la production de certains matérieux pour vous éviter des heures de récolte.

=> Enfin, le but ultime est de battre L'Ender Dragon, le boss final, après avoir visité toutes les dimensions du jeu et avoir récolté les objets vous permettants d'accéder au monde hostile dans lequel ce trouve L'Ender Dragon. Une fois en face de lui, accrochez-vous, un combat éprouvant vous attend! Une fois battu, vous pourrez vous battre contre d'autres boss comme le Whither ou le Warden, encore plus puissants que L'Ender Dragon. Et voilà, vous avez "terminé" le jeu. En réalité pas vraiment, il vous reste à créer une infinité de structures, à voyager, à défendre vos amis villageois contre leurs agresseurs ou encore à vous battre une nouvelle fois contre ces boss


Le créatif


=> Dans ce mode, vous pouvez tout faire, c'est le mode bac à sable par excellence. Des blocks illimités, pas besoin de se défendre des montres, pas besoin de se nourrie, tout ce dont vous aurez besoin sera votre créativité!

=> Pour vous faciliter la tache, Minecraft vous permet de voler et de vous déplacer plus vite qu'en survie, votre expérience de jeu n'en sera que plus fluide. De nombreux raccourcis sont aussi mis à votre disposition pour que vous puissiez construire plus facilement.

=> Le mode créatif de Minecraft est là pour que vous laissiez libre cours à votre imagination. Si vous êtes de mauvaise humeur ou que vous vous ennuyez, vous pouvez faire exploser de la TNT partout pour vous défouler. Vous pouvez evidemment construire des structures toujours plus extravagantes, rejoindre des projets multijoueurs par le biais de serveurs pour recréer des villes, planètes et galaxies entières!!

=> Recréez les décors de vos films et séries préférés, construisez vos héros préférés et mettez les en scène dans le décor de votre choix! Vous pouvez tout créer de A à Z mais vous avez aussi la possibilité d'importer le travail d'autres joueurs grâce à des sites répertoriant les mondes d'autres joueur qui veulent le partager et le mettre à la disponibilité d'autres joueurs. Alors à vous de jouer! N'hésitez pas à nous envoyer vos maps via ce lien!


Le hardcor


=> Le mode hardcor est un mode de jeu qui vous permet de jouer en survie mais avec une difficulté accrue. En effet, si vous mourez, votre partie est terminée. De plus, il es impossible de revenir en arrière, vous ne pouvez pas revenir à la dernière sauvegarde. Vous êtes donc obligé de faire attention à tout ce que vous faites. Si vous mourrez, vous perdrez tout ce que vous avez construit, tout ce que vous avez récolté, tout ce que vous avez fabriqué. Vous devrez donc recommencer à zéro.


Le mode spéctateur


=> Le mode spectateur est un mode de jeu qui vous permet de regarder les autres joueurs jouer. Vous pouvez les suivre, les observer, les filmer, les photographier. Eux ne pourront pas vous voir. Vous pouvez aussi vous téléporter à n'importe quel endroit de la map, mais aussi passser à travers les murs et les blocs. En revanche il vous est impossible de casser des blocks ou encore d'en poser.
